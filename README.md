# www

Source code for my official website.

## Get Started

### Prerequisites

- Bun 1.0+

### Install Dependencies

Run:

```sh
bun install
```

### Start Development Server

Run:

```sh
bun run dev
```

### Build for Production

Run:

```sh
bun run build
```

### Preview Production Build

Run:

```sh
bun run preview
```

### Code Quality Checks

Run:

```sh
bun run check
bun run lint
```
